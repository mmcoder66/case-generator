# 审计追踪参考用例

本文件用于维护 `configuration_manage/audit_trail` 菜单的只读参考用例。新增参考用例时使用下方标准表头。

优先级以 `generation_rules/priority_rules.md` 和 `generation_rules/audit_trail_coverage_rules.md` 为准：PRD 明确要求的审计详情字段准确性可按合规发布阻断项标为 `P0`；详情导出、搜索筛选和普通展示体验默认不因涉及审计追踪自动升为 `P0`。审计追踪权限统一按审计追踪菜单权限理解。
