# 边界值用例模板

## 适用范围

适用于验证系统中的输入长度、文件大小、数量、周期范围、阶段数、字段类型、数据范围、阈值、日期区间、意见长度等边界场景。

## 编写要求

- 明确边界值来源。
- 至少覆盖最小值、最大值、低于最小值、高于最大值。
- 预期结果必须说明系统允许、拦截或提示的具体表现。
- 涉及业务计算或处理时，应说明结果、阻断状态、字段清理或错误提示是否符合规则。
