| 导出入口和按钮 | 工艺能力汇总页面右上角增加导出按钮，受权限控制 | 工艺能力汇总页面显示导出按钮；无导出权限时导出按钮不显示；无工艺能力汇总查看权限时模块入口不可见 |
| 同步实时导出 | 点击导出后同步生成 PDF，前端显示加载提示，不允许重复点击 | 点击导出显示加载提示并阻止重复点击；导出成功浏览器自动下载 PDF 文件 |
| 文件命名规则 | 文件名格式为 {方案名称}_{导出监控项目数量}_{时间戳}.pdf | 【待确认】导出 PDF 文件名格式正确 |
| 导出内容 - 页眉 | 页眉显示方案名称、导出时间、产品信息 | 【待确认】导出 PDF 页眉显示方案名称、导出时间和产品信息 |
| 导出内容 - 表格列 | 表格列包含监控项目名称、监控类型、正态性、CPK、PPK、超规格限百分比、异常批次、方案名称、操作（仅文字） | 导出 PDF 表格列内容完整 |
| 导出内容 - 异常批次 | 异常批次显示具体批次号，无异常显示"无" | 异常批次为空时 PDF 显示"无" |
| 导出内容 - 附注 | 表格后附 CPK/PPK 颜色图例、规格限说明 | PDF 附注显示 CPK/PPK 颜色图例和规格限说明 |
| 导出数据完整性 | 导出当前方案下所有监控项目，不受分页限制 | 导出包含当前方案所有监控项目，不受分页限制 |
| 导出视图无关性 | 导出不依赖连接视图或矩阵视图，统一列表视图格式 | 导出不依赖视图切换，统一列表视图格式 |
| 性能上限 | 数据量超过 500 条阻止导出并提示 | 数据量超过 500 条阻止导出 |
| 异常 - 无数据 | 当前方案无监控项目时提示暂无数据 | 当前方案无监控项目时提示暂无数据 |
| 异常 - 网络中断 | 网络中断时提示检查网络 | 【待确认】网络中断时提示导出失败请检查网络 |
| 异常 - 服务器异常 | 服务器异常或生成失败时提示重试 | 【待确认】服务器异常或生成失败时提示导出失败请重试 |
| 权限控制 | 导出按钮受工艺能力汇总导出权限控制 | 无导出权限时导出按钮不显示 |
| 审计追踪 - 列表记录 | 导出成功后审计追踪列表新增导出记录 | 导出成功后审计追踪列表新增导出记录 |
| 审计追踪 - 详情字段 | 详情包含基本信息、导出环境、参数列表、文件信息、监控项目信息列表 | 查看导出审计追踪详情字段正确 |
| 审计追踪 - 操作类型下拉 | 工艺能力汇总模块操作类型下拉新增导出选项 | 审计追踪操作类型下拉新增导出选项 |
| UI 图缺失 | 工艺能力汇总导出 UI 图未提供，按钮位置、加载提示样式、错误弹窗样式等视觉细节按 PRD 文本覆盖 | UI 图补充后再细化：按钮精确位置、加载遮罩视觉样式、错误弹窗交互细节 |
